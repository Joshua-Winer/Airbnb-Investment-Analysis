# Airbnb-Investment-Analysis
# Airbnb Investment Analysis Using Python

## Objective
Identify property characteristics associated with higher Airbnb cash-on-cash returns using regression analysis.

## Methodology
- Data cleaning using pandas
- Exploratory analysis
- Regression modeling
- Visualization with matplotlib

## Key Findings
- Bedroom count positively impacts return
- Higher prices reduce ROI
- Mid-priced multi-bedroom properties show strongest performance

## Tools
Python, pandas, and matplotlib
